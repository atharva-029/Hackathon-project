# Hackathon-project
Kisan Mitra is a web-based platform designed to help farmers with useful agricultural information in one place. It provides weather updates, crop details, market prices, government schemes, and fertilizer/pesticide guidance in both Hindi and English.
This project was developed as part of a hackathon to support digital farming and improve access to information for farmers.

Features

Weather updates with charts and 3D visualization

Mandi (market) price information

Crop information and crop guide

Fertilizer & pesticide recommendations

Government schemes section

AI assistant (chat & voice support)

Hindi and English language support

Responsive design for mobile and desktop
